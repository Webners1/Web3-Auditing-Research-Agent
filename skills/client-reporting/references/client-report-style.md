# Client Report Style Guide

Write for founders, operators, and clients who need clarity fast.

## Style

- direct, professional, and calm
- specific over generic
- no filler
- no hype language
- no unexplained jargon if a simpler phrase works

## Good Client Report Behavior

- start with the executive view
- put the most consequential risks first
- make recommendations actionable
- explain trade-offs without hedging away the conclusion
- separate confirmed facts from interpretation

## Avoid

- academic essay tone
- vague `best practice` advice with no implementation path
- too much protocol trivia in the executive summary
- unsupported market predictions
