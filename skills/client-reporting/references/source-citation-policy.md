# Source Citation Policy

Every report that depends on external information must include exact source links.

## Required Behavior

- cite the exact URL, not just the homepage name
- prefer official docs, official sites, official governance, explorers, and curated ecosystem data sources
- if a metric came from DefiLlama or L2Beat, say so explicitly
- if a conclusion is an inference from multiple sources, say it is an inference

## Minimum Source Expectations By Report Type

- Security report:
  - code under review
  - tool outputs
  - official protocol docs if needed
  - explorer or deployment links if addresses matter

- Architecture report:
  - official protocol docs
  - benchmark protocol docs
  - chain or ecosystem docs
  - curated data sources where relevant

- Business proposition report:
  - official product docs
  - DefiLlama and/or L2Beat where relevant
  - official chain docs or roadmap sources
  - any benchmark protocol docs used for comparison

- Diligence report:
  - source appendix covering all major inputs

## Last Resort Rule

Generic web search is a fallback only after official and curated registry sources are checked first.
