# Protocol Memory Schema

Use this schema when creating or updating protocol memory files.

---

## `profile.md`

```markdown
# [Protocol Name]

- Slug:
- Category:
- Stage:
- Chains:
- Repositories:
- Frontend:
- Core contracts:
- Important dependencies:
- Revenue model:

```

## `working-memory.md`

```markdown
# Working Memory

## Current State
- [Latest validated facts]

## Active Hypotheses
- [What seems true but is not fully proven]

## Session Notes
- [Dated notes]
```

## `findings.md`

```markdown
# Findings

## Confirmed
- [Severity] [title] - [status]

## Leads
- [title] - [why it is still unconfirmed]
```

```markdown

```

## `open-questions.md`

```markdown

```
